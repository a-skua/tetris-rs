# Tetris

To try it out:
[GitHub pages](https://a-skua.github.io/tetris-rs)

## Building

You need the following libraries to build:

- [Rust](https://www.rust-lang.org/)
  - [wasm-pack](https://rustwasm.github.io/wasm-pack/)
- [Deeno](https://deno.land/)

### Build and Run

```sh
cd wasm
make build serve
```
